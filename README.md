# Music School

A modern music education platform built with **Next.js 15**, **Tailwind CSS v4**, and **Aceternity UI**. Features course listings, instructor profiles, upcoming webinars, testimonials, and a contact page — with full Light/Dark theme support.

## Tech Stack

- Next.js 15 (App Router)
- Tailwind CSS v4
- Aceternity UI components
- Framer Motion
- TypeScript

## Local Setup

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
